How to use the Executor

URL Format:

executor.html?form=<config-url>

Example:

executor.html?form=https://raw.githubusercontent.com/user/repo/main/form-config.json

Requirements:

  • Valid JSON configuration URL
  • CORS-enabled endpoint (e.g., GitHub raw URLs)
  • Configuration must include fields array
